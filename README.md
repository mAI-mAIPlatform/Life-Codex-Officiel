# Life Codex Officiel

Démo d'un moteur de jeu TypeScript/React avec:
- boucle de jeu à pas fixe;
- streaming de chunks avec priorités;
- architecture workers (physics/pathfinding/procgen);
- interface **Liquid Glass**.

## Démarrer

```bash
npm install
npm run dev
```

## Vérification

```bash
npm run check
npm run build
```
